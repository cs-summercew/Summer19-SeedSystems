Churro is a stack-based interpreted programming language in which the code is entirely made up of churros. Design Principles. Churros are magical and should therefore be the only syntax element. The language must include mutable state in order to irk Calvin Beck. Language Concepts. Churro operates on two structures: a stack and an array.
